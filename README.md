# Automated Power line Inspection
this repository presents a comparative deep learning study for automated power line inspection using drone imagery.
The project explores and evaluates multiple models to identify the most effective approach for detecting faults in real-world conditions, with a focus on performance optimization and reliability.
It includes an end-to-end pipeline covering data preprocessing, model training, evaluation, and analysis.
